Etapas del desarrollo del software

Durante la construcción del software se define una serie de etapas que permiten cumplir exitosamente con el objetivo de los requerimientos.

Análisis: En esta etapa se conocen las necesidades y se dimensiona lo que se quiere construir, se define el alcance del proyecto, quienes lo van a utilizar y de qué manera. En esta etapa se debe tener una definición clara de lo que se quiere obtener ya que a partir de unos buenos requerimientos depende del éxito y mantenibilidad del software.

Diseño: En esta etapa se debe determinar a partir de los requerimientos analizados en la fase anterior, detalladamente la funcionalidad y comportamiento del software con los usuarios u otros sistemas; se define interfaz del sistema, modelo de base de datos, definición de documentación.

Codificación: Codificar lo plasmado en la etapa de diseño.

Implementación: En esta etapa se realiza proceso de instalación, pruebas de calidad del software y capacitación del software a los usuarios, con el fin de verificar el cumplimiento de la necesidad.

Mantenimiento: Se evalúan faltantes, revisión de problemas durante la ejecución del software, soporte al cliente.
